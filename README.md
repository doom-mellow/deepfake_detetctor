# Deepfake Detector 🔍

A multi-modal AI-based detection system that analyzes different types of media to identify whether content is real or AI-generated.

---

## 🚀 Features

* 🖼️ Image Deepfake Detection
* 🎥 Video Deepfake Detection
* 🎧 Audio Analysis
* 📝 Text AI Detection
* 📤 Upload files or use URLs
* ⚡ Fast and interactive UI

---

## 🛠️ Tech Stack

* React + TypeScript
* Vite
* Tailwind CSS
* Supabase (optional backend)

---

## 📂 Project Structure

* `src/components` → UI components
* `src/pages` → Application pages
* `src/services` → Detection logic
* `src/types` → Type definitions

---

## ▶️ How to Run Locally

```bash
npm install
npm run dev
```

---

## 🧠 How It Works

The system performs forensic-style analysis:

* Image → checks GAN artifacts, lighting inconsistencies
* Video → detects blinking anomalies & frame inconsistencies
* Audio → analyzes spectral patterns & voice naturalness
* Text → evaluates structure, perplexity, and patterns
